## 二、搬瓦工支持的全部付款方式：支付宝到底在不在列

先把这个最基础的问题摆桌面。搬瓦工目前支持四种主流支付方式：

- **Alipay（支付宝）**——国内用户首选，扫码即付，到账快

## 三、支付宝付款完整流程：从注册到扫码，一步一步走

下面把整个流程拆开讲，每一步都说人话。

> 小提醒：邮箱别用太冷门的，找回密码、续费提醒、补货通知都得靠它。Gmail、QQ 邮箱都行。

### 第二步：选套餐，加入购物车

这一步是关键中的关键。在 VPS 套餐页里挑一个你想要的配置，点 "Order Now"。下一页会让你选 **Billing Cycle（计费周期）** 和 **Location（机房）**。

选完点 "Continue" 进入结算页。

### 第三步：核对订单 + 填优惠码

结算页会显示订单详情、价格、可用支付方式。**这里有个省钱小动作别漏**：在 "Promotional Code" 输入框里填优惠码，点 "Validate Code"，页面会刷新显示折后价。搬瓦工的优惠码是循环生效的，新购和续费都能用。

### 第四步：选支付方式，扫码完成付款

在支付方式栏里勾选 **Alipay**，点 "Complete Order"。这时会跳到 Payssion 网关，弹出一个支付宝扫码页面。

打开支付宝 App，扫屏幕上的二维码，确认金额，付款。

完成之后会自动跳回搬瓦工后台，你会看到订单状态变成 "Paid"，VPS 也几乎同时开通——通常几分钟内就能在 "Services" 里看到你的服务器，IP、root 密码都会发到你邮箱。

## 四、支付宝付款失败的常见原因和解决方法

新手最容易在这一步崩心态。根据社区反馈，搬瓦工支付宝付款失败大概有这么几种情况：

**1. 扫码页面打不开 / 一直转圈**

这是最常见的一种。原因是 Payssion 网关偶尔会抽风，国内网络访问它有时不稳。解决方法很简单：换个浏览器试试（Chrome、Edge 各试一遍），或者刷新重试几次。实在不行，**临时切换到银联或云闪付扫码付款**，效果一样。

**2. 提示 SECONDARY_MERCHANT_STATUS_ERROR**

这种报错通常和商户状态有关，跟你的账户没关系。一般过几分钟重试就能解决，或者换一种支付方式过渡。

**3. 扫了码但支付没成功**

可能是支付宝侧的延迟，或者金额超了你的单笔限额。先在支付宝里看交易记录，确认没扣款再重试。如果扣了款但订单没生效，把支付宝账单截图发邮件给搬瓦工客服（support@bandwagonhost.com），他们处理得还挺快。

**4. 优惠码填了但价格没变**

确认你点过 "Validate Code" 按钮。光填不点是不会生效的。另外有些限量版套餐不支持优惠码，下单前留意说明。

## 八、几个新手常踩的小坑

最后再补几个细节，都是别人用钱试出来的经验：

1. **首单别选太长的计费周期**。月付或季付先试一个月，确认线路和速度满意再续长周期，避免一次性砸大钱发现不合适。

2. **续费要自己手动发起**。搬瓦工不自动续费，到期前会发邮件提醒，但邮件可能被你忽略，建议在日历上加个提醒。

3. **限量版套餐要看清库存**。THE PLAN、BiggerBox、MiniChicken 这类限量版经常补货又下架，下单前先看 [stock.bwg.net](https://bit.ly/BandwagonHost) 的实时库存。已购用户通常能续费，新用户不一定买得到。

4. **支付宝付款失败先别慌**。先在支付宝里确认没扣款，再换浏览器或换支付方式重试。绝大多数情况都能解决。
